### 👋 Welcome to Props3D Github repo
We love creating and building interactive props and replicas from our personal favorites like Aliens, Dredd 2012, or The Mandalorian. This has also led us on a whole new venture creating our Blamcon lightguns for retro and modern gaming. Here you'll find free code repos used for some of our existing props.
